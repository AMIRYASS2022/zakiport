# Portfolio Images

Place your work images in this folder to use them in the portfolio section.

## How to add your images:

1. Copy your work images from "C:\Users\yassi\Downloads\mes trabajos" to this folder
2. Rename them with descriptive names (e.g., "brand-design-1.jpg", "photography-1.jpg", "web-design-1.jpg")
3. Update the portfolioItems array in src/components/Portfolio.tsx to reference your images

## Image naming suggestions:
- design-*.jpg (for design work)
- photography-*.jpg (for photography)
- web-*.jpg (for web projects)
- ai-*.jpg (for AI projects)

## Example structure:
```
public/portfolio/
├── design-1.jpg
├── design-2.jpg
├── photography-1.jpg
├── photography-2.jpg
├── web-1.jpg
└── ai-1.jpg
```
